# Cite Me - Front End

This is the front-end application for the Cite Me project, built using modern web technologies.

## Usage

The project uses [pnpm](https://pnpm.io/) as its package manager. To get started, follow these steps:

The project requires Node.js version 16 or higher, and Node.js 22 is recommended. To install Node.js, you can use [nvm](https://github.com/nvm-sh/nvm).

1. Install pnpm

```bash
npm install -g pnpm
```

2. Install dependencies

```bash
pnpm install
```

3. Start the development server

```bash
pnpm dev
```

4. Open your browser and navigate to `http://localhost:3000` to view the application.
